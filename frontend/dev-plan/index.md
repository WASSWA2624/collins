## Delivery rule

Ship the simplest safe ICU workflow first: Home, Admit, Tracking, ABG / Vent Update, live summary, and offline draft/sync status. Reviewer, dataset, admin, and model screens should remain role-gated and built only after the core clinical workflow is reliable.
