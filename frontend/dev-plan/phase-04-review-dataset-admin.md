# Phase 4 — Review, Dataset Capture, and Admin Dashboards

## Goal

Support specialist review, safe dataset capture, and facility/platform oversight.

## Scope

- Add role-gated Review Queue.
- Add approve, request correction, exclude, and conflict-resolution UI.
- Add Dataset Capture screen for pasted ICU notes.
- Show editable structured preview with uncertain/missing fields highlighted.
- Submit parser outputs for review, not training.
- Add facility admin dashboard metrics.
- Add platform admin dashboard metrics.
- Add approved dataset export UI only for approved research/data governance roles.

## Done when

- Normal clinicians cannot access reviewer/admin-only details.
- Unreviewed dataset captures cannot appear as training-approved.
- Review actions are clear and auditable.
- Dashboard cards match the app write-up.
